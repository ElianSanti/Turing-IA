Las funcionalidades de Tableau Server Management garantizan escalabilidad, disponibilidad, capacidad de administración y seguridad para simplificar los procesos de administración. Satisfaga sus requisitos específicos de implementación y configuración. Aproveche la experiencia y las inversiones en tecnología existentes, y optimice su entorno para alcanzar un verdadero análisis de autoservicio escalable.

Brinde datos de alta calidad con Tableau Catalog

Tableau Catalog, el catálogo de análisis nativo, brinda una vista completa de las fuentes y los activos de datos. Además, lo ayuda a comprender la dependencia de los datos y proporciona metadatos en contexto para un descubrimiento más rápido. Con el objetivo de ayudar a mejorar la toma de decisiones, se comparten detalles de los datos, incluidas las advertencias de calidad, justo donde se consumen.

Mantenga los datos actualizadas con Tableau Prep Conductor

Tableau Prep Conductor crea un entorno con datos preparados y listos para usar, disponibles para todos. Automatice cómo programa, supervisa y administra flujos de datos para siempre tener datos actualizados disponibles para el análisis.

Rediseñe la gobernanza de datos con las conexiones virtuales

Las conexiones virtuales son la clave para compartir datos seguros fácilmente en Tableau. Mejore los datos a lo largo de todo el ciclo de vida con políticas coherentes para todos los usos. De esta manera podrá asegurarse de que se comporten cómo usted quiere, en especial en el caso de los datos confidenciales.
